Red Vs Blue Attack Defense Simulator

Red Side:- Attacker
2 different forms 
a)attacks within a single virtual machine and uses a DVWA as a target to attack
b) attack occuring between 2 systems, in this case a mac laptop and a windows laptop using a virtual machine. It also uses a DVWA on the victim pc(MAC)
The tools used for attacking are:- Hydra,SQLMAP,Burpsuite,Ping Sweeps,MSFvenom,theHarvester,gobuster

Blue Side:-Defense
2 different forms
a)defense within a single virtual machine where the DVWA is made safe and the attacks are blocked 
b)Defense between 2 systems 
Tools used are Suricata,Splunk
